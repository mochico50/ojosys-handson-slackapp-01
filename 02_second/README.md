# 第2回：リアクションデータを取得してスプレッドシートに記録しよう 📊

## 今日のゴール

自分のチャンネルのリアクション数を集計して、**スプレッドシートに書き込まれる**

```
Slack API（リアクションデータを取得）
　↓
GAS（投稿者ごとに集計）
　↓
スプレッドシート（書き込む）
```

---

| 時間 | 内容 |
|---|---|
| 5分 | 前回の復習・今日のゴール確認 |

## 前回のおさらい

前回作ったもの：
```
Slack App（/ranking-{名前} を受け取る）
　↓　Request URL
GAS（「受け取ったよ！」と返す）
```

今回はGASの中身を本格的にしていきます！

---

## STEP 1：スプレッドシートを準備する

### 1-1. スプレッドシートを新規作成する

1. [Google スプレッドシート](https://sheets.google.com) を開く
2. **「新しいスプレッドシートを作成」** をクリック
3. タイトルを `ranking-{自分の名前}` に変更する 例）`ranking-mochico`

### 1-2. スプレッドシートのIDをコピーする

スプレッドシートのURLを確認する

```
https://docs.google.com/spreadsheets/d/【ここがID】/edit
```

> 📋 このIDをコピーしておく（あとでGASに貼り付けます）

---

## STEP 2：Slack APIトークンを確認する

前回インストール時にコピーした **Bot User OAuth Token** を使います。

> 📋 `xoxb-`から始まる文字列です  
> 忘れた場合：[Slack API](https://api.slack.com/apps) → 自分のApp → 「Install App」から確認できます

---

## STEP 3：チャンネルIDを確認する

### 3-1. チャンネルIDの調べ方

1. Slackで自分のチャンネル（`#ranking-{自分の名前}`）を右クリック
2. **「チャンネル詳細を表示」** をクリック
3. 一番下までスクロールするとチャンネルIDが表示される（`C`から始まる文字列）

> 📋 このIDをコピーしておく

---

## STEP 4：AIにコードを書いてもらう

AIツールを開いて、以下を**そのままコピペして送信**する  
（`【】`の中は自分の情報に書き換えてください）

```
Google Apps Scriptで、以下の処理をするコードを書いてください。

処理の流れ：
1. Slackのスラッシュコマンド（doPost）を受け取る
2. Slack APIを使って特定チャンネルの投稿を取得する
3. 各投稿についた❤️リアクションの数を投稿者ごとに集計する
4. 集計結果をGoogleスプレッドシートに書き込む
5. Slackに「集計完了！スプレッドシートを確認してね」と返す

設定値：
- SLACK_TOKEN：【Bot User OAuth Tokenをここに貼る】
- CHANNEL_ID：【チャンネルIDをここに貼る】
- SPREADSHEET_ID：【スプレッドシートIDをここに貼る】
- 集計するリアクション：❤️（heart）

条件：
- 設定値は定数として冒頭にまとめる
- コードにコメントをつけて説明してください
```

> 💡 AIが出してくれたコードをそのままコピーしてOKです

---

## STEP 5：GASを更新する

### 5-1. 前回のGASを開く

[Google Apps Script](https://script.google.com) を開いて、前回作った `ranking-bot` を開く

### 5-2. コードを貼り替える

1. エディタの中身を**全部消す**
2. AIが書いてくれたコードを**貼り付ける**
3. `Ctrl + S` で保存

### 5-3. 再デプロイする

コードを変えたので再デプロイが必要です！

1. 右上の **「デプロイ」** ボタンをクリック
2. **「デプロイを管理」** をクリック
3. 鉛筆アイコン✏️をクリック
4. バージョンを **「新しいバージョン」** に変更
5. **「デプロイ」** をクリック

> ⚠️ 「新しいデプロイ」ではなく **「デプロイを管理」** から更新してください  
> 新しいデプロイを作るとURLが変わってしまいます

> ✅ デプロイが完了したらOK（URLは変わっていないはず）

---

## STEP 6：動作確認する

### 6-1. テスト用のリアクションをつける

1. 自分のチャンネル（`#ranking-{自分の名前}`）に何か投稿する
2. その投稿に ❤️ リアクションをつける
3. 他の人に頼んで ❤️ をつけてもらうとより楽しい！

### 6-2. コマンドを実行する

1. `/ranking-{自分の名前}` と入力して送信
2. 「集計完了！スプレッドシートを確認してね」と返ってきたらOK

### 6-3. スプレッドシートを確認する

スプレッドシートを開いて、データが書き込まれていることを確認する

> ✅ 投稿者名とリアクション数が記録されていたら**今日のゴール達成です🎉**
