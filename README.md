# お嬢シス SlackApp ハンズオン　第1回 🎉

## このハンズオンでつくるもの

Slackのスラッシュコマンドを打つと、  
**一番リアクションをもらった人のランキングがスプレッドシートに記録されるボット**を作ります！

```
/ranking-{自分の名前}
　↓
特定チャンネルのリアクションを集計
　↓
スプレッドシートにランキングを記録 🎉
```

> 💡 コードは全部AIに書いてもらいます。コーディング経験は不要です！

---

## 対象者

- コーディング未経験の方
- SlackAppを使った開発をしたことがない方
- IT従事者・情シス部隊の方

---

## 全体の流れ

| | テーマ | ゴール |
|---|---|---|
| 事前準備 | 環境構築 | ツールの準備をする |
| 第1回 | つなぐ | `/ranking-{名前}` を打ったらGASが反応する |
| 第2回 | とってくる | リアクション数がスプレッドシートに書き込まれる |
| 第3回 | 完成！ | ランキングがスプレッドシートに記録される 🎉 |

---

## 技術スタック

| ツール | 役割 |
|---|---|
| Slack App | スラッシュコマンドを受け取る |
| Google Apps Script（GAS） | 処理・集計をする |
| Googleスプレッドシート | ランキングを記録する |
| AIツール | コードを書いてもらう |

---

## AIツールについて

コードはAIに書いてもらいます。以下のどれかを使ってください。

- [Claude](https://claude.ai)
- [ChatGPT](https://chatgpt.com)
- [Gemini](https://gemini.google.com)（Googleアカウントがあればそのまま使えます）
