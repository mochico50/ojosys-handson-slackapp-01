# 発展課題：一番リアクションした人も出してみよう 🚀

## 概要

ハンズオンでは「一番リアクションをもらった人」を集計しました。  
今度は自分で「一番リアクションをした人」も追加してみましょう！

---

## ヒント

### データの取り方の違い

| | もらった人 | した人 |
|---|---|---|
| 何を見る | 投稿についたリアクション数 | リアクションしたユーザーリスト |
| どこにある | `reactions[].count` | `reactions[].users[]` |

Slack APIのレスポンスには、リアクションした**ユーザーのリスト**も含まれています。

```json
"reactions": [
  {
    "name": "heart",
    "count": 3,
    "users": ["U12345", "U67890", "U11111"]
  }
]
```

この `users` の配列を全投稿分集計すると「一番リアクションした人」がわかります！

---

## AIへのプロンプト例

```
前回作ったGoogle Apps Scriptのコードに機能を追加してください。

追加してほしいこと：
「一番リアクションをした人ランキング」も集計してスプレッドシートに追加する

ヒント：
- Slack APIのレスポンスのreactions[].usersに、
  リアクションしたユーザーのIDリストが入っています
- ユーザーIDからユーザー名を取得するには
  Slack APIの users.info を使います

スプレッドシートの構成：
- シート1：リアクションをもらった人ランキング（既存）
- シート2：リアクションをした人ランキング（新規追加）
```

---

## できたらやってみよう

- ランキングをSlackにも投稿する
