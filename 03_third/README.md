## STEP 1：AIにコードを書いてもらう

AIツールを開いて、以下を**そのままコピペして送信**する  
（前回のコードがあれば一緒に貼り付けるとより精度が上がります）

```
前回作ったGoogle Apps Scriptのコードを改良してください。

追加してほしいこと：
1. 集計結果を❤️リアクション数の多い順に並べ替える
2. スプレッドシートに以下の形式で書き込む
   - A列：順位（🥇🥈🥉4位5位...）
   - B列：投稿者名
   - C列：リアクション数
   - D列：集計日時
3. 書き込む前にシートの内容をクリアして上書きする
4. 1位〜3位は絵文字（🥇🥈🥉）で表示する

条件：
- SLACK_TOKEN、CHANNEL_ID、SPREADSHEET_IDは引き続き
  PropertiesService.getScriptProperties().getProperty("プロパティ名") で取得する
- コードにコメントをつけて説明してください
```

> 💡 前回うまくいったコードをAIに貼り付けて「改良して」と頼むのがポイントです！

---

## STEP 2：GASを更新する

### 2-1. コードを貼り替える

1. GASエディタの中身を**全部消す**
2. AIが書いてくれたコードを**貼り付ける**
3. `Ctrl + S` で保存

### 2-2. 再デプロイする

1. 右上の **「デプロイ」** ボタンをクリック
2. **「デプロイを管理」** をクリック
3. 鉛筆アイコン✏️をクリック
4. バージョンを **「新しいバージョン」** に変更
5. **「デプロイ」** をクリック

> ⚠️ 必ず **「デプロイを管理」** から更新してください（URLが変わらないように）

---

## STEP 3：動作確認する

### 3-1. コマンドを実行する

1. 自分のチャンネル（`#ranking-{自分の名前}`）を開く
2. `/ranking-{自分の名前}` と入力して送信

### 3-2. スプレッドシートを確認する

スプレッドシートを開いて、ランキングが記録されていることを確認する

```
期待する結果：
```

> ✅ ランキング形式でスプレッドシートに記録されたら**完成です🎉🎉🎉**
